# AI要約SNS - 構想

## 一言で言うと

**投稿すると自動的にAIが3行に要約してくれるSNS**

---

## 解決したい課題

- SNSの投稿は長くなりがち。全部読むのは大変
- かといって短文制限（Twitter/X方式）だと表現が制約される
- 「書く人は自由に長く書ける」「読む人はまず要約で概要を掴める」を両立したい

## コアコンセプト

1. ユーザーは好きなだけ長文を書ける（最大10,000文字）
2. 投稿した瞬間、AIが自動で **3行に要約** する
3. タイムラインには **要約のみ** が表示される
4. 興味があれば原文を展開して読める

```
┌─────────────────────────────────┐
│ @username                       │
│                                 │
│ ・AIが生成した要約1行目          │
│ ・AIが生成した要約2行目          │
│ ・AIが生成した要約3行目          │
│                                 │
│ [原文を見る]         2026/03/28 │
└─────────────────────────────────┘
```

## 特徴

- **書き手**: 文字数を気にせず自由に書ける
- **読み手**: 3行で要点を把握できる。気になったら原文へ
- **AI**: 投稿のたびに自動要約。ユーザー操作は不要

## ユーザー体験の流れ

1. ユーザー名 + パスワードだけでアカウント作成
2. テキストを書いて「投稿」を押す
3. AIが3行要約を生成（バックグラウンド）
4. タイムラインに要約付きで表示される
5. `/:noteId` で個別の投稿を共有できる

## 画面

| パス              | 内容                                            |
| ----------------- | ----------------------------------------------- |
| `/`               | タイムライン（要約付き投稿一覧 + 投稿フォーム） |
| `/:noteId`        | 投稿の個別表示                                  |
| `/login`          | ログイン                                        |
| `/register`       | アカウント登録（ユーザー名 + パスワードのみ）   |
| `/user/:username` | ユーザーの投稿一覧                              |

## 技術スタック

- **Frontend**: Vue 3 + Vite + Tailwind CSS
- **Backend**: Cloudflare Workers + D1 (SQLite)
- **API**: oRPC（Contract First で型安全）
- **AI**: Cloudflare Workers AI
- **認証**: ユーザー名 + パスワード → JWT

## 開発アプローチ

**Contract First** — API の型定義（契約）を最初に書き、サーバーとクライアントがそれに従う。
契約が Single Source of Truth。
